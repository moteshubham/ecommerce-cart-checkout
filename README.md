# Ecommerce Cart & Checkout

Ecommerce cart and checkout system with React frontend and Node.js/Express backend.

## Overview

This project implements a shopping cart system with checkout functionality, coupon management, and admin reporting.

## Getting Started

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Features

- Shopping cart management
- Checkout with coupon validation
- Admin coupon generation
- Admin reporting
